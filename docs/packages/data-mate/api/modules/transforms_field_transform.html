truncate

▸ truncate(input: StringInput, _parentContext: unknown, args: { size: number }): string | string[] | null

Will truncate the input to the length of the size given if given an array it will convert everything in the array excluding null/undefined values

example

FieldTransform.truncate('thisisalongstring', {}, { size: 4 }); // 'this'; FieldTransform.truncate(['hello', null, 'world'], {}, { size: 2 }); // ['he', 'wo'];
