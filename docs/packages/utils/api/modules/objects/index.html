isSame

▸ isSame(input: unknown, target: unknown): boolean

Compares two values and returns a boolean if they are the same. Object keys are sorted before comparison, arrays are NOT sorted and are compared value for value

example isSame({ key1: 1, key2: 2 }, { key2: 2, key1: 1 }) === true; isSame(null, null) === true; isSame(undefined, undefined) === true; isSame(NaN, NaN) === true; isSame(3, 3) === true isSame('hello', 'hello') === true isSame([1, 2, 3], [1, 2, 3]) === true isSame([{ some: 'obj' }], [{ some: 'obj' }]) === true

isSame(undefined, null) === false; isSame([1, 2, 3], [1, 3, 2]) === false isSame([1, 2, 3], [1, 2, undefined, 3]) === false isSame(true, 'true') === false;
